## 4. Phase 2 사전 준비 (라이브러리 세팅)
다음 주차부터 진행될 엑셀 기반 대량 작업, 서버 사이드 제어, 그리고 알림 시스템 연동을 위해 다음 패키지들의 설치 및 조사를 완료했습니다.
- **SheetJS (xlsx)**: 클라이언트/서버 단의 엑셀(`.xlsx`) 파일 파싱 및 다운로드를 위해 `apps/web`에 설치 완료 (`npm install xlsx`).
- **firebase-admin**: Cloud Functions에서 권한 제한 없이 Firestore를 제어하고 트랜잭션을 처리하기 위해 최신 버전 설치 완료 (`npm install firebase-admin`).
- **카카오 알림톡 SDK 조사**: 
  - (예정) 알리고(Aligo) 또는 솔라피(Solapi) 등 서드파티 알림톡 API 연동 방식 검토 진행.
  - (구조) 캠페인 상태 변경(선발, 반려 등) 시 Cloud Functions `onCreate`/`onUpdate` 트리거를 활용해 알림톡 발송 API를 비동기 호출하도록 아키텍처 초안 설계 완료.
